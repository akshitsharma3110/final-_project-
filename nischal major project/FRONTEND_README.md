# Disaster Image Classifier - Web Frontend

## Setup

1. Install Flask dependencies:
```bash
pip install -r requirements_frontend.txt
```

2. Run the Flask app:
```bash
python app.py
```

3. Open browser and go to:
```
http://localhost:5000
```

## Features

- Upload disaster images (JPG, PNG)
- Real-time prediction with 95.43% accuracy
- Confidence score visualization
- All class predictions displayed
- Drag & drop support
- Mobile responsive design

## Supported Disaster Classes

1. Biological and Chemical Pandemic
2. Cyclone
3. Drought
4. Earthquake
5. Flood
6. Landslide
7. Tsunami
8. Wildfire

## Model Performance

- Overall Accuracy: 95.43%
- Precision: 95.51%
- Recall: 95.43%
- F1-Score: 95.42%
